- 先要有php(我是安裝[wamp][wamp server])

- 為方便將php路徑加入path，以便直接使用php指令。
```
C:\wamp\bin\php\php5.5.12
```

- 下載 [phpunit.phar] 至php資料夾
```
C:\wamp\bin\php\php5.5.12
```

- 執行phpunit
```
php phpunit.phar
```
