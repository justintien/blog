


##FacebookJavaScriptLoginHelper

if use this class, frontend javascript FB_init function must set the params

######[FacebookJavaScriptLoginHelper_ref1]

```js
FB.init({
  appId      : 'APP-ID',
  xfbml      : true,
  version    : 'v2.1',
  //important
  cookie     : true
});
```
